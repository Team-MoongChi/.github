## Background

### 현황

- 1인 가구수 증가
- 경제적 · 구매 효율성 한계
- '지역기반' 커뮤니티 필요성

### 기존 서비스 한계 및 차별화 전략

1. 불신 → 리더게이지, 팔로워게이지로 신뢰도를 측정가능하도록 수치화
2. 실제 1인 가구 만을 위한 서비스 부족 → AI, LLM을 활용한 맞춤 상품 추천
